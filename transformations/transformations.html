Showing Change with Visual Clues - Transformations

There are four distinct transform properties

Transformations

The transform property applies a 2D or 3D transformation to an element. This allows you to manipulate elements with a triggered event like hover.


  • rotate
  • scale
  • skew
  • translate

The most basic form of code for these properties will look something like this:


selector { transform: rotate(20deg);}

you will always use transform: followed by one of the four properties; this is followed by a set of () with the value inside. Each property's values are a little different
