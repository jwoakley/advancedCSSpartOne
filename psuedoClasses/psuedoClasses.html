Showing Change with Visual Clues - Psuedo Classes

Psuedo Classes

Psuedo Classes Define a Special State on an Element.

We are all already aware of psuedo classes, even if we don't know what they are are. Some of the most common examples are listed here:


  • link
  • visited
  • hover
  • active

The code for psuedo classes will always look something like this:


a:hover { color: #0000FF;}

selector : psuedoClass


one thing to note:
there is NO SPACE between the selector, the colon and the psuedo class; this is VERY important
